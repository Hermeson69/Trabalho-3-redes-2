================================================================================
COMO EXECUTAR O PROJETO
================================================================================

1. Iniciar os containers:
   $ docker-compose up -d --build

2. Verificar que tudo está rodando:
   $ docker-compose ps

3. Executar os testes de carga:
   $ docker exec load_client python3 /app/load_test.py

4. Ver resultados:
   $ cat resultados/resultados_testes.txt
   $ cat resultados/resumo_executivo.txt

5. Ver logs dos servidores Python:
   $ docker logs nginx_server
   $ docker logs apache_server

6. Acessar interfaces web:
   - Prometheus: http://localhost:9090
   - Grafana: http://localhost:3000 (admin/admin)

7. Parar tudo:
   $ docker-compose down
